nn – Nearest Neighbour calculator¶

PDist¶

Computes pairwise distances, fast implementation.

Currently, ‘cdist()’ function from scipy is the fastest, although it uses only one core. Using np.sum(np.power(dist,2),2) is 10x slower.

class nn.pdist.PDist[source]¶

Implementation of pairwise distance calculation.

Distance function is set to squared euclidean. No results for other functions yet, but they can do as well.

Eats something like 30% of the total image classification time. Fast parallel BLAS or GPU implementation would be beneficial.
