Features

  • automatic analysis and classification of images
  • scalable parallel execution
  • fast binary cSIFT extractor
  • fast and convenient HDF5 database
  • keep all the data
  • readable module structure

Overview

WIBC is an automatized image classification toolbox written primarly in Python. It is designed for web image classification, but suitable for arbitrary image data.

The author is working in Aalto University School of Science in collaboration with F-Secure corp in Espoo, Finland.

This is an open source software licensed under GPL v2, started in 2012.
