**WIBC** is an automatized image classification toolbox written primarly in Python.
It is designed for web image classification, but suitable for arbitrary image data.

The author is working in `Aalto University School of Science <http://sci.aalto.fi/en/>`_ in collaboration with `F-Secure corp <http://www.f-secure.com/fi/web/home_fi/home>`_ in Espoo, Finland.

This is an open source software licensed under **GPL v2**, started in 2012.
